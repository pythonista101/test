Kaggle verwacht dat we onzelf authenticeren voordat de API gebruikt kan worden. Hiervoor wordt eerst de kaggle api token ingevoerd.
Vervolgens wordt een kaggle.json file van kaggle gedownload.Een .kaggle folder moet aangemaakt worden in de hoofdmap gebruikers.
Een kaggle.json file wordt van downloads naar de hoofdmap verplaatst.Daarna vindt initalisatie plaats (zie code).
Er zijn twee soorten datasets die gedownload kunnen worden. Een competitiedataset of de op zichzelf staande datasets.
Wij maken gebruik van twee datasets die op zichzelf staand zijn.Deze gaan niet gepaard met een wedstrijd en kunnen door iedereen geupload worden.
