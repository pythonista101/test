Authenticatie kaggle Api
Kaggle verwacht dat we onszelf authenticeren voordat de API gebruikt kan worden.

Stel je kaggle api token in
download kaggle.json file van kaggle
maak een .kaggle folder aan in hoofdmap gebruikers
Verplaats de kaggle.json file van downloads naar hoofdmao

vanaf dan kan kaggle import werken vanuit jupiter
