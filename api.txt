Om de API toe te passen op Kaggle wordt verwacht dat je jezelf eerst authenticeert.

Er zijn twee soorten datasets die gedownload kunnen worden. Een competitiedataset of de op zichzelf staande datasets. Wij maken gebruik van twee datasets die op zichzelf staand zijn.
